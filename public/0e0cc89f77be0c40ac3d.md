## openコマンドの基本構文

```bash
open [オプション] [ファイル/ディレクトリ/URL]
```

## -jオプション：アプリケーションを隠して起動

`-j`オプション（`--hide`）は、アプリケーションを隠れた状態（非表示）で起動します。アプリケーションは実行されますが、ウィンドウが表示されません。

```bash
# 計算機を隠れた状態で起動
open -j -a "Calculator"
```
