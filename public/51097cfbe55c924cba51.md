Dynamic ImportsでローディングUIを設定するには以下のように記述します。

```jsx
import dynamic from 'next/dynamic'
 
const Sample = dynamic(
  () => import('../components/Sample'),
  {
    loading: () => <p>Loading...</p>,
  }
)
 
export default function Page() {
  return (
    <div>
      <Sample />
    </div>
  )
}

```
