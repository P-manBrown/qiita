## 設定方法

以下のコマンドを実行すると`gs branch submit`などを実行した際に既存のブランチのみが更新されるようになります。

```
# リポジトリレベル
git config --local spice.submit.updateOnly true

# グローバルレベル
git config --global spice.submit.updateOnly true
```
