## 設定方法

`settings.json`の言語設定に以下を追加して、LSP折り畳みを有効にします（v0.224.0以降）。

```json
"document_folding_ranges": "on"
```

## 動作仕様

| 項目 | 説明 |
|------|------|
| **デフォルト状態** | `"document_folding_ranges": "off"` |
| **有効時の動作** | Tree-sitterの折り畳みを**無効化**し、LSPの折り畳みを使用 |
| **フォールバック** | LSPが結果を返さない場合、自動的にTree-sitterに戻る |
