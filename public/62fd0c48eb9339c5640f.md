リスト内でフラグメントを使う際、省略記法 `<>...</>` では `key` を渡せません。

```jsx
// ❌ keyが設定できない
{items.map(item => 
  <>
    <h1>{item.title}</h1>
    <p>{item.body}</p>
  </>
)}
```

`Fragment` を明示的にインポートして使用します。

```jsx
import { Fragment } from 'react';

// ✅ keyを渡せる
{items.map(item => 
  <Fragment key={item.id}>
    <h1>{item.title}</h1>
    <p>{item.body}</p>
  </Fragment>
)}
```
