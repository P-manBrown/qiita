```bash
gs stack submit --fill
```

`--fill`フラグを使用すると、既存のコミットメッセージから自動的にPRのタイトルと本文を入力します。

```bash
# 良いコミットメッセージの例
git commit -m "Fix user login validation

- Add email format validation
- Improve error messages
- Update unit tests"

# このコミットから自動でPRが作成される
gs stack submit --fill
```
