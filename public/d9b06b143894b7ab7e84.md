| キー | 説明                 |
|------|---------------------|
| h    | 左に移動            |
| j    | 下に移動            |
| k    | 上に移動            |
| l    | 右に移動            |
| w    | 単語分先へ移動      |
| b    | 単語分前へ移動      |
| e    | 単語の末尾へ移動    |
| W    | 空白区切りの単語分先へ移動 |
| B    | 空白区切りの単語分前へ移動 |
| E    | 空白区切りの単語の末尾へ移動 |
| 0    | 行頭へ移動          |
| ^    | 空白以外の行頭へ移動 |
| $    | 行末へ移動          |
| gg   | ファイル先頭に移動    |
| G    | ファイル末尾に移動    |
