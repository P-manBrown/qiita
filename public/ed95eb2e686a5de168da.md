## 設定方法

`settings.json`に以下の設定を追記します。

```jsonc:settings.json
{
    "lsp": {
      "vtsls": {
        "settings": {
          "typescript": {
            "tsserver": {
              "experimental": {
                "enableProjectDiagnostics": true
              }
            },
            "updateImportsOnFileMove": {
              "enabled": "always"
            }
          }
        }
      }
    },
}
```
