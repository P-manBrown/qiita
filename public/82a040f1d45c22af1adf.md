### 設定方法

`settings.json` で隠しファイルのパターンをカスタマイズできます。

```json
{
  "hidden_files": ["**/.*", "**/*.tmp", "**/build/**"]
}
```

**設定例**

```jsonc
{
  "hidden_files": [
    "**/.*",
    "**/*.tmp",
    "**/build/**",
    "**/node_modules/**",
    "**/*.log"
  ]
}
```
