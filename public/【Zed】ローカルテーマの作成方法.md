## ローカルテーマとは

Zedエディタでは、拡張機能として公開せずに個人用のカスタムテーマを作成できます。

## 保存場所

**macOS / Linux:**
```
~/.config/zed/themes/
```

**Windows:**
```
%USERPROFILE%\AppData\Roaming\Zed\themes\
```
## テーマの適用

1. Zedを再起動
2. `Cmd+K Cmd+T` (macOS) / `Ctrl+K Ctrl+T` (Windows/Linux) でテーマセレクターを開く
3. 作成したテーマを選択
