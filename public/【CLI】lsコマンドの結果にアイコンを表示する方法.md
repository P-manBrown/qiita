## ezaでアイコンを表示

### インストール

```bash
brew install eza
```

### アイコン表示

```bash
# アイコンを表示
eza --icons

```

## lsdでアイコンを表示

### インストール

```bash
brew install lsd
```

### アイコン表示

```bash
# デフォルトでアイコンが表示される
lsd
```
