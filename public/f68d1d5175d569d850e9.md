## 設定方法

`settings.json`に以下を追加します。

```json
{
  "vim": {
    "use_system_clipboard": "never"
  }
}
```

## use_system_clipboardの設定値

| 値 | 説明 |
|---|---|
| `"always"` | 常にシステムクリップボードを使用（デフォルト） |
| `"never"` | システムクリップボードを使用せず、Vimレジスタのみ |
| `"on_yank"` | ヤンク時のみシステムクリップボードにコピー |
