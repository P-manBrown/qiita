## 方法

角括弧記法での任意の値をサポートします。

```css
@utility tab-* {
  tab-size: --value([integer]);
}
```

## 例
```html
<div class="tab-[1]">1文字のタブサイズ</div>
<div class="tab-[8]">8文字のタブサイズ</div>
<div class="tab-[2.5rem]">2.5remのタブサイズ</div>
```

任意の型を許可する場合は`--value([*])`を使用します。
