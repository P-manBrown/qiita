## gsからgit-spiceに変更される

`gs`は今後使用できなくなり、`git-spice`に変更されます。

https://github.com/abhinav/git-spice/releases/tag/v0.24.0

## `gs`を使い続ける方法

Shellの設定に以下を追加することで`gs`を使用し続けられます。

```
alias gs='git-spice'
```
