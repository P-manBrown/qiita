ターミナルで以下のコマンドを実行し、複製元のリポジトリをクローンします。

```terminal
git clone --bare https://github.com/UserName/old-repo.git
```

複製先のリポジトリを作成し、以下のコマンドを実行します。

```terminal
$ cd old-repo.git
$ git push --mirror https://github.com/UserName/new-repo.git
```

不要なファイルを削除します。

```terminal
cd ..
rm -rf old-repo.git
```
