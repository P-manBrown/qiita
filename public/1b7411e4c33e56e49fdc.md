## 設定方法

### 1. キーバインドの設定（必須）

`keymap.json` に以下を追加します（v0.224.0以降）。

```json
{
  "context": "Workspace",
  "bindings": {
    "cmd-i": "tab_switcher::OpenInActivePane"
  }
}
```

キーは好みに応じて変更してください。

### 2. タブバーを非表示にする（任意）

Vim/Helix にはタブバーがないため、同様のUXにしたい場合は `settings.json` に追加します。

```json
{
  "tab_bar": {
    "show": false
  }
}
```

### 3. タブを閉じるときに隣のペインへ移動する（任意）

Vim のウィンドウクローズ（`:q`）に近い動作にしたい場合は、`keymap.json` に追加します。

```json
{
  "context": "Pane",
  "bindings": {
    "cmd-w": "pane::JoinIntoNext"
  }
}
```

これにより、タブを閉じる際にアイテムが破棄されず、隣のペインに移動します。

## 既存の `ToggleAll` との違い

| | `tab_switcher::ToggleAll` | `tab_switcher::OpenInActivePane` |
|---|---|---|
| 表示対象 | アクティブペインのタブのみ | 全ペインのタブ（重複なし） |
| 開く場所 | 選択したファイルが存在するペイン | 常にアクティブなペイン |
| 主な用途 | 通常のタブ管理 | Vim/Helix風のバッファ管理 |
