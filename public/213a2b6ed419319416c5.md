Appliteを使用することでHomebrewのCaskをGUIで管理できるようになります。

https://github.com/milanvarady/Applite

インストールするには以下のコマンドを実行します。

```terminal
brew install applite
```

以下のような画面でCaskの検索、インストール、更新、アンインストールができます。
