## `Capitalize<StringType>`

文字列の先頭の文字を大文字に変換するには以下のように記述します。

```typescript
type LowercaseGreeting = "hello, world";
type Greeting = Capitalize<LowercaseGreeting>;
        
// Greeting = "Hello, world"
```
