## 設定方法

`settings.json`に以下を追記するとVimモードインジケーターの色を変更できます。

```jsonc:settings.json
{
  "theme_overrides": {
    "テーマ": {
      // Vimモードインジケーターのテキスト色
      "vim.mode.text": "#1a1b26",

      // Normalモード時のインジケーター背景色(紫)
      "vim.normal.background": "#bb9af7",
      
      // Insertモード時のインジケーター背景色(赤)
      "vim.insert.background": "#f7768e",
      
      // Replaceモード時のインジケーター背景色(オレンジ)
      "vim.replace.background": "#e0af68",
      
      // Visual(文字選択)モード時のインジケーター背景色(青)
      "vim.visual.background": "#7aa2f7",
      
      // Visual Line(行選択)モード時のインジケーター背景色(シアン)
      "vim.visual_line.background": "#7dcfff",
      
      // Visual Block(矩形選択)モード時のインジケーター背景色(緑)
      "vim.visual_block.background": "#73daca"
    }
  }
}
```
