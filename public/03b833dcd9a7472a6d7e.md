## jqを使う方法

```bash
$ echo -n "テスト" | jq -sRr '@uri'
%E3%83%86%E3%82%B9%E3%83%88
```

- `-s` (slurp): 入力全体を1つの文字列として読み込む
- `-R` (raw-input): 入力をJSON文字列ではなく生のテキストとして扱う
- `-r` (raw-output): 出力時にJSONの引用符を削除
- `@uri`: URLエンコード用のフォーマット関数

### jqのインストール

jqは以下のコマンドでインストールできます。

```bash
brew install jq
```
