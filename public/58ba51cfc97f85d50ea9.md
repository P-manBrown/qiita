## ホスト名を取得するコマンド

以下のコマンドでホスト名が取得できます。

### `hostname` コマンド

```terminal
hostname
```

```terminal
$ hostname
MacBook-Pro-13-inch.local
```



## `uname -n` コマンド

`uname`でもホスト名を取得できます。

```terminal
uname -n
```

```terminal
$ uname -n
MacBook-Pro-13-inch.local
```
