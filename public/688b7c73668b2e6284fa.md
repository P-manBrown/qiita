## 設定方法

`settings.json`に以下のように記述します。

```json
{
  "whitespace_map": {
    "space": "•",
    "tab": "→"
  }
}

```

- `space`: 通常のスペース文字の表示に使用する文字（デフォルト: `·`）
- `tab`: タブ文字の表示に使用する文字（デフォルト: `→`）
