`gh-dash`を使用するとターミナル上でGitHubのIssueやPRを表示できます。

https://github.com/dlvhdr/gh-dash

## インストール

`gh`コマンドをインストール

```terminal
brew install gh
```

`gh-dash`をインストール

```terminal
gh extension install dlvhdr/gh-dash
```

## 使い方

以下のコマンドを使用するとIssueやPRの検索画面が表示されます。

```terminal 
gh dash
```
