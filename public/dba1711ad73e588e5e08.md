## 正規表現一覧
```
.	任意の1文字に合致
*	直前の文字が0回以上繰り返されている場合に合致
+	直前の文字が1回以上繰り返されている場合に合致
?	直前の文字が０個か１個の場合に合致（最長一致）
*?	直前の文字が0回以上繰り返されている場合に合致（最短一致）
+?	直前の文字が1回以上繰り返されている場合に合致（最短一致）
??	直前の文字が０個か１個の場合に合致（最短一致）
[  ]	[ ]の中のどれか1文字に合致
[^  ]	[ ]の中の文字以外に合致
( ～ | … )	～か…かどちらかに合致
{n}	直前の文字の桁を指定
\w	アルファベット、数字、アンダスコア（ _ ）のどれかの1文字
\W	アルファベット、数字、アンダスコア以外の1文字
\d	数字1文字
\D	数字以外の1文字
\s	半角スペース、タブ、改行のどれか1文字
\S	半角スペース、タブ、改行以外の1文字
(  )	( )内を1文字扱い
^	行頭
$	行末
\<	単語の先頭
\>	単語の末尾
\b	単語の先頭か末尾
\B	単語の先頭、末尾以外
\A	ファイルの先頭
\z	ファイルの末尾
\G	直前の一致文字列の末尾
\0	一致した文字列全体
\l	次の1文字を小文字にする
\L ～ \E	挟まれた文字列を小文字にする
\u	次の1文字を大文字にする
\U ～ \E	挟まれた文字列を大文字にする
\n	改行
\t	タブ
```
